Developer documentation
#######################

This developer documentation contains the following sections:

1. :ref:`devdoc-get-started` explains how you can start developing code and
   documentation;
2. The :ref:`devdoc-architecture` section explains how you can find your way
   around the source code, and how the different modules/languages API interact
   with one another.
3. The :ref:`devdoc-versions` section explains how we manage the version numbers
   of all the packages in the metatensor repository.
4. :ref:`devdoc-local-ci` explains how to run the GitHub Actions pipeline on
   your local machine for faster debugging.
5. :ref:`devdoc-faq` covers some common questsions and workflows.
